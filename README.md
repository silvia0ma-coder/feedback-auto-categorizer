# feedback-auto-categorizer
Automated customer feedback routing workflow —  routes submissions to the right Slack channel  in under 60 seconds using n8n, Lovable &amp; Google Sheets

![Workflow Diagram](workflow-diagram)

## 🚀 How to use
1. Open the [workflow.json](workflow.json) file in this repo.
2. Copy the code.
3. Paste it into your n8n canvas.
